# Rúbrica general del proyecto

| Criterio | Excelente | Adecuado | Básico | Insuficiente |
| --- | --- | --- | --- | --- |
| Comprensión del problema | Explica con claridad la necesidad del invernadero y justifica las variables medidas. | Identifica el problema y las variables principales. | Reconoce el contexto, pero con justificación limitada. | No relaciona el sistema con el problema del invernadero. |
| Diseño electrónico | El circuito es coherente, está bien documentado y justifica componentes y conexiones. | El circuito funciona y recoge los componentes principales. | El circuito es parcial o necesita ayuda frecuente. | El diseño no funciona o no se entiende. |
| Simulación | Prueba distintos escenarios, registra resultados y corrige errores. | Simula el sistema principal y comprueba su funcionamiento. | Realiza pruebas básicas sin análisis suficiente. | No presenta simulación funcional. |
| Programación | El código está estructurado, comentado y responde correctamente a los sensores. | El código funciona y permite leer sensores y activar salidas. | El código funciona parcialmente o con poca explicación. | El código no funciona o no se comprende. |
| Montaje físico | Montaje ordenado, seguro y coherente con el esquema. | Montaje funcional con errores menores. | Montaje parcial o dependiente de ayuda constante. | Montaje inseguro, incompleto o no funcional. |
| Trabajo en equipo | Reparto equilibrado, comunicación eficaz y registro de decisiones. | El equipo se organiza y cumple la mayoría de tareas. | Hay reparto desigual o coordinación irregular. | No hay colaboración efectiva. |
| Memoria técnica | Completa, clara, con esquemas, código, pruebas, errores y mejoras. | Incluye los apartados principales. | Presenta información incompleta o poco justificada. | No documenta adecuadamente el proceso. |
| Exposición oral | Clara, rigurosa, con demostración y respuesta solvente a preguntas. | Explica el proyecto y muestra evidencias suficientes. | Presentación superficial o poco organizada. | No comunica el funcionamiento del proyecto. |
