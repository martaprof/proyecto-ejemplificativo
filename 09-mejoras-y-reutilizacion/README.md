# Mejoras y reutilización

Esta carpeta documenta posibles mejoras, adaptaciones a otros niveles, variantes del proyecto y recomendaciones para que otros docentes puedan reutilizarlo.

## Posibles mejoras técnicas

- Sustituir el potenciómetro por un sensor real de humedad.
- Incorporar una pantalla LCD para mostrar lecturas.
- Registrar datos en una tarjeta SD.
- Añadir comunicación inalámbrica.
- Conectar el sistema a una plataforma IoT.
- Usar sensores más precisos de temperatura y humedad.
- Incorporar actuadores reales: ventilador, bomba de agua o sistema de sombreado.

## Adaptaciones didácticas

- Reducir el proyecto a una simulación completa para centros sin laboratorio.
- Dividir el proyecto en prácticas independientes.
- Usarlo como proyecto final de trimestre.
- Convertirlo en reto interdisciplinar con Biología o Ciencias Ambientales.
- Simplificar la programación mediante bloques en Tinkercad.
- Ampliar la memoria técnica para alumnado con mayor nivel.

## Reutilización

Otro docente puede reutilizar el proyecto modificando:

- nivel educativo;
- variables ambientales medidas;
- componentes disponibles;
- duración;
- grado de autonomía del alumnado;
- formato del producto final;
- criterios de evaluación.

## Variante simplificada solo con Tinkercad

Para centros sin material físico, el proyecto puede realizarse íntegramente en Tinkercad:

- mantener el sistema de medición y avisos como producto final;
- sustituir el montaje físico por capturas y tabla de validación;
- evaluar la simulación, el código, la memoria técnica y la exposición;
- usar el potenciómetro como entrada manipulable para simular humedad;
- dejar el servomotor como ampliación opcional.

## Variante ampliada

Para grupos con más autonomía o más tiempo, se pueden añadir:

- sensor real de humedad;
- pantalla LCD;
- registro de datos;
- comunicación inalámbrica;
- ventilador o bomba de agua como actuador real;
- comparación entre datos simulados y datos medidos.

## Variante interdisciplinar

El proyecto puede conectarse con:

- Biología, analizando necesidades de distintas plantas;
- Ciencias Ambientales, estudiando sostenibilidad y eficiencia;
- Matemáticas, trabajando tablas, gráficas y ajuste de umbrales;
- Digitalización, incorporando registro de datos o visualización remota.

## Pendientes

- Pendiente de completar tras la primera implementación real en aula: tiempos, dificultades, ajustes de umbrales y mejoras de materiales.
