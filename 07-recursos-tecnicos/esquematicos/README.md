# Esquemáticos

Esta carpeta recoge los esquemas exportados desde Tinkercad o generados como referencia técnica para el proyecto.

## Archivos disponibles

| Archivo | Uso dentro del proyecto |
| --- | --- |
| [`sistema-medicion-invernadero.pdf`](sistema-medicion-invernadero.pdf) | Esquemático de referencia para el sistema de medición y avisos. |
| [`etapa-alimentacion.pdf`](etapa-alimentacion.pdf) | Esquemático de referencia para la etapa de alimentación. |
| [`control-servomotor-seguimiento.pdf`](control-servomotor-seguimiento.pdf) | Esquemático de referencia para el control del servomotor mediante sensores de luz. |

## Uso didáctico

Estos documentos pueden emplearse como apoyo para:

- interpretar el paso de la simulación al montaje físico;
- comprobar conexiones antes de alimentar el circuito;
- documentar la memoria técnica del alumnado;
- comparar el diseño inicial con el montaje final;
- introducir la lectura de esquemas eléctricos en el aula.
