# Sesiones

Esta carpeta organiza el desarrollo del proyecto por sesiones, fases y bloques de conocimiento.

## Planificación general

El proyecto se integra en la materia Tecnología e Ingeniería I como una actividad de desarrollo progresivo. No es necesario realizar todas las sesiones de forma consecutiva; pueden distribuirse a lo largo del curso y vincularse a los bloques de saberes correspondientes.

La duración estimada es de **21 sesiones presenciales de 55 minutos**, combinadas con trabajo autónomo del alumnado fuera del aula.

Cada README de sesión combina dos usos: planificación docente y material de lectura para el alumnado. Por ello, además de propósito, contenidos, evidencias y recursos, cada sesión incluye una explicación de los contenidos, un desarrollo guiado paso a paso, un ejemplo guiado y mini-ejercicios prácticos o teóricos.

## Secuencia metodológica común

Cada bloque técnico puede seguir esta estructura:

1. Sesión explicativa en el aula.
2. Tarea individual de cálculo, diseño o programación.
3. Corrección y retroalimentación.
4. Simulación en Tinkercad.
5. Aplicación al sistema del invernadero.
