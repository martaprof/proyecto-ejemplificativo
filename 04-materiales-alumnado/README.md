# Materiales para el alumnado

Esta carpeta incluye los recursos que se entregarían al alumnado: guiones de práctica, instrucciones, retos, fichas de trabajo, enlaces y documentos de apoyo.

## Reto del proyecto

Un invernadero necesita mantener unas condiciones adecuadas de luz, temperatura y humedad para favorecer el crecimiento de las plantas. El reto consiste en diseñar un sistema electrónico que permita medir algunas de esas condiciones y avisar cuando se encuentren fuera de un intervalo adecuado.

El sistema deberá poder simularse en Tinkercad y, si el material del aula lo permite, montarse físicamente con Arduino.

## Entregables del alumnado

Cada equipo deberá entregar:

- esquema o esquemas del circuito;
- simulación en Tinkercad;
- código Arduino;
- registro de pruebas;
- memoria técnica;
- presentación final;
- reflexión sobre mejoras posibles.
- commits o entregas parciales con mensajes claros.
